# Mood Color 🎨

Type any text and get a color palette generated from the mood and sentiment.

![Screenshot](screenshot.png)

## Features

- **Sentiment analysis** — Keywords, emoji detection, and punctuation patterns
- **19 distinct moods** — Happy, sad, angry, calm, creative, nostalgic, and more
- **Dynamic palettes** — 5 harmonious colors generated per mood
- **Copy to clipboard** — Click any swatch or chip to copy the hex code
- **Adaptive background** — Gradient shifts to match the detected mood
- **Real-time updates** — Palette regenerates as you type

## How It Works

The analyzer scores your text against keyword dictionaries and emoji mappings. The dominant mood drives HSL-based palette generation with controlled randomness for variety.

## Usage

Open `index.html` in any modern browser. No build step required.
